### 🧑‍💻 About

I'm an SDE at **Amazon**, where I design and deliver backend services for a multi-region distributed platform — writing production code in Java, TypeScript, and Python across microservices, serverless functions, and data pipelines. I lead design reviews, build developer tooling and test automation, and run primary oncall for production services.

Previously at **ServiceNow** (4 years), I shipped full-stack features and built test automation frameworks and CI tooling used across release pipelines. I hold an **M.S. in Computer Science** from **Florida State University**, where I focused on distributed systems, parallel programming, and OS-level optimizations, and I represented my university at the **ACM-ICPC** regional finals (Amritapuri + Gwalior).

I'm passionate about systems that are fast, reliable, and elegant.
